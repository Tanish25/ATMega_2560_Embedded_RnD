Registers involved:

GPIO:
DDRL: Data Direction Register for Port L= 0xFE for input (Port L- pin 49)
DDRC: Data Direction Register for Port C= 0x01 for output (Port C- pin 37)

PINL: Port L Data storage, data is read from this register

PORTL: Port L Data Register, enabling/disabling pull-up resistors
PORTC: Port C Data Register, enabling/disabling pull-up resistors
